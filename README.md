## 📖 简介
**星驰引擎_禁用LTPO** 是专为 **欧加（Oplus）真设备** 打造的高刷模块，支持原生刷新率以及超频 DTBO 功能。
该模块能够强行锁定用户希望的刷新率档位，无论是在原生使用还是在超频环境下，都能实现稳定高效的屏幕刷新控制。

---

## ✨ 功能特性

### 🛡️ 刷新率锁定
*   **支持原生刷新率**：如 60Hz、90Hz、120Hz 等。
*   **支持超频 DTBO**：可根据需求自定义刷新率档位。
*   **强制锁定**：锁定指定档位，避免系统自动降频或切换。

### ⚡ 超频 DTBO 兼容性
*   **自动检测**：刷入后会自动运行所有检测到的刷新率档位。
*   **必须保留 120Hz**：超频 DTBO 中 **必须保留 120Hz 档位**，否则可能导致功能失效或不稳定。
*   **故障排查**：当超频 DTBO 用户自定义应用切换功能不生效时，请优先联系提供该 DTBO 的作者更新至最新版本，以修复潜在的监测 BUG。

## 🛠️ 常见问题与解决方案

### 防锁屏与关机重启问题
*   **问题描述**：
    *   在某些设备上，超频 DTBO 在锁屏或关机重启后无法直接切换到高档位。
    *   原生用户在关机重启后，几秒内可自动切换到之前设定的档位。
*   **解决方案**：
    *   建议更新超频 DTBO 至最新版本。

### 💡 关于配置保留与管理器
*   **覆盖安装注意**：模块覆盖安装 **无法保留** 所有设置。
*   **如何备份配置**：如果要保留自定义应用配置，请提前备份模块目录下的 `apps.conf` 文件，安装完新版本后覆盖回原位即可。
*   **管理器推荐**：
    *   **KernelSU** 及其分支体验最好。
    *   **Magisk** 用户必须下载 **KsuWebUI** 软件，并授予 Root 权限后使用。

## 🚀 使用步骤

1.  **下载并刷入**：下载「星驰引擎_禁用LTPO」模块并在面具/KSU中刷入。
2.  **重启设备**：系统将自动检测所有可用刷新率档位。
3.  **配置档位**：
    *   进入 WebUI 管理器进行手动配置。
    *   选择需要锁定的刷新率档位。
4.  **检查确认**：确保超频 DTBO 中保留了 120Hz 档位，以保证功能完整性。
