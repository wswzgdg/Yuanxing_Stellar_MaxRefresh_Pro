### 星驰引擎_禁用LTPO 是专为欧加真设备打造的高刷模块，支持原生刷新率以及超频DTBO功能。该模块能够强行锁定用户希望的刷新率档位，无论是在原生使用还是在超频环境下，都能实现稳定高效的屏幕刷新控制。

### 功能特性
​
- 1. 刷新率锁定​
     支持原生刷新率（如60Hz、90Hz、120Hz等）。
     支持超频DTBO功能，可根据需求自定义刷新率档位。
     强制锁定指定档位，避免系统自动降频或切换。
- 2. 超频DTBO兼容性​
     当超频DTBO用户自定义应用切换功能不生效时，请优先联系提供该DTBO的作者，更新至最新版本，以修复潜在的监测BUG。
- 3. 自动检测与配置​
     刷入后会自动运行所有检测到的刷新率档位。
     重启后需手动进入管理器进行配置，确保设置生效。
- 4. 超频DTBO要求​
     必须保留120Hz档位，否则可能导致功能失效或不稳定。

### 防锁屏与关机重启问题
​
- 1. 问题描述​
     在某些设备上，超频DTBO在锁屏或关机重启后无法直接切换到高档位。
     原生用户在关机重启后，几秒内可自动切换到之前设定的档位。
- 2. 解决方案​
     建议更新超频DTBO至最新版本，模块覆盖安装无法保留所有设置，如果要保留自定义应用，可以提前备份模块目录下的apps.conf，覆盖文件就可以了，kernelsu以及分支体验最好，Magsik必须下载KsuWebUI软件，给root权限使用

### 使用步骤
- 1. 下载并刷入星驰引擎_禁用LTPO模块。
  2. 重启设备，系统将自动检测所有可用刷新率档位。
  3. 进入管理器进行手动配置，选择需要锁定的刷新率档位。
  4. 确保超频DTBO中保留120Hz档位，以保证功能完整性。
